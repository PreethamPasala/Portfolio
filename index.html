AI Pong

The game of Pong is an excellent example of a simple RL task. On the low level the game works as follows: we receive an image frame (a 210*160*3) byte array (integers from 0 to 255 giving pixel values)) and we get to decide if we want to move the paddle UP or DOWN (i.e. a binary choice). After every single choice the game simulator executes the action and gives us a reward: Either a +1 reward if the ball went past the opponent, a -1 reward if we missed the ball, or 0 otherwise. And of course, our goal is to move the paddle so that we get lots of reward.

CSRF and Privilege escalation attacks

This project sheds light on a number of existing tools that can be used to test for the CSRF vulnerability. The main objective of the research is to identify the available solutions to prevent CSRF attacks. By analysing the techniques employed in each of the solutions, the optimal tool can be identified.

PARALLEL AND DISTRIBUTED IMAGE ENCRYPTION USING DES

The project proposes and implements a method to implement a task in its entirety and to parallel it across distributed systems. The project therefore uses image encryption for the same purpose as a testing and benchmarking algorithm.
